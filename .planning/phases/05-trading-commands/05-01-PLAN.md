# Phase 5: Trading Commands

> Source: docs/TRADING_ARCHITECTURE.md — Step 6

## Goal

Telegram commands for interacting with the trading system — portfolio view, market snapshots, trade history, mode switching, emergency controls.

## Tasks

### Task 1: Create `bot/handlers/trading_cmds.py`
New router with commands:

- `/portfolio` — show balance, open positions, daily P&L. Calls `trading.get_balance()` + `trading.get_positions()` + `trading.get_daily_pnl()`
- `/market [pair]` — market snapshot with indicators. Defaults to BTC/USDT. Calls `market.get_market_summary([pair])`. Include latest Kronos + Chronos predictions if available.
- `/trades [n]` — last N trades (default 10). Calls `trading.get_trade_history(n)`. Show reasoning for each.
- `/mode paper|live` — switch trading mode. `live` requires confirmation ("Sei sicuro? Scrivi CONFERMA"). Calls `trading.set_mode(mode)`.
- `/kill` — emergency close all positions. Calls `trading.emergency_close_all()`. Immediate, no confirmation needed.
- `/autonomous on|off` — enable/disable autonomous trading. When on, the market scanner executes trades without confirmation.
- `/scan` — force immediate market scan: fetch latest data, run Kronos/Chronos predictions, compute indicators, pass everything to Claude for analysis. Claude responds with market assessment and trade recommendation.

### Task 2: Register router in `bot/main.py`
- Import router from `bot.handlers.trading_cmds`
- `dp.include_router(trading_cmds.router)` — before messages.router (same pattern as kronos_cmds)

### Task 3: Graceful degradation
If `TRADING_ENABLED=false`, commands should respond with "Trading non abilitato." instead of crashing.
If execution layer isn't ready (Kraken CLI not installed), `/portfolio` and `/trades` should say so.

## Verification

- `/portfolio` shows paper balance and positions (or "no positions")
- `/market BTC/USDT` shows price, RSI, EMA, MACD, Bollinger, Kronos/Chronos predictions
- `/trades` shows trade history or "nessun trade"
- `/mode live` asks for confirmation, doesn't switch without it
- `/kill` closes all positions immediately
- `/autonomous on` enables autonomous mode
- `/scan` triggers Claude analysis with full market context
- All commands require authorized user
